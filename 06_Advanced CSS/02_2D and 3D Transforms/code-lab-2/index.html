The transition property

To create a transition effect, you must specify the CSS property you want to add a transition to, and the duration of the transition.

The CSS transition property is a shorthand property for:

  • transition-property (Required)
  • transition-duration (Required)
  • transition-timing-function
  • transition-delay

The transition is triggered when there is a change in the element's properties.

This often happens within pseudo-classes (:hover, :active, :focus, or :checked).

Transition Speed Curve

The transition-timing-function property specifies the speed curve of the transition effect.

This property can have one of the following values:

linear - transition will keep the same speed from start to end

linear

ease - transition will start slow, then go fast, and end slow (this is default)

ease

ease-in - transition will start slow

ease-in

ease-out - transition will end slow

ease-out

ease-in-out - transition will have a slow start and end

ease-in-out

cubic-bezier(n,n,n,n) - lets you define your own values in a cubic-bezier function

A Cubic Bezier curve is defined by four points P0, P1, P2, and P3. P0 and P3 are the start and the end of the curve and, in CSS these points are fixed as the coordinates are ratios. P0 is (0, 0) and represents the initial time and the initial state, P3 is (1, 1) and represents the final time and the final state. The cubic-bezier() function can be used with the animation-timing-function property and the transition-timing-function property.

cubic-bezier(0,0,1,1)

cubic-bezier(0.25,0.1,0.25,1)

cubic-bezier(0.42,0,1,1)

cubic-bezier(0,0,0.58,1)

cubic-bezier(0.42,0,0.58,1)
